Reporting List Selection

Items to be matched in a search are selected from the lists provided.

  1. Clicking on an item in a list selects and highlights that item.
  2. Selecting the first blank item deselects the list from a search
  3. If a multi-line list is shown more than one item can be selected.
    • a range of items can be selected by clicking on one item and then clicking on a second item with the shift key held down. The range of items between the two items will be selected and highlighted.
    • a list of items can be selected by clicking on one item and then clicking on other items with the Ctrl key held down. A maximum of 10 individual items are allowed.
    • Please Note:
      • The only valid search modifiers for a selected range/list are = and <>.
      • If any other modifier is selected = will be assumed.
      • If only one item is selected then the full range of modifiers are available.
  4. The symbolic drop-down lists modify the search field.
    =Find EXACT match for...text
    <Find value LESS THAN...text
    >Find value GREATER THAN...text
    <=Find value LESS THAN or EQUAL TO...text
    >=Find value GREATER THAN or EQUAL TO...text
    <>Find NOT EQUAL TO...text
    nlFind NULL (no text entered)
    i.e. This searches for the absence of data in this field
  5. Only lists with selected items are used for searching, except for the nl option.
