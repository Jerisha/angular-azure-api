Reporting Help Index

The reporting option screen consists of one or more of the following 3 sections.
Click on a Title (link) for more detailed information.

  1. Wildcard Selection

    2. Search text can be entered in the text boxes provided.
    • Searches can be case sensitive.
    • The drop-down lists modify the search field.
    • Empty text boxes do not take part in a search.
  2. List Selection

    3. This is similar to the Wildcard selection section with drop-down lists provided instead of text boxes.
  3. Displayed Columns

    4. Ticking a check box will add this column to the report produced by pressing the search or excel buttons.
  4. Page Control

    5. The following report parameters are set from this section.
    • The maximum rows (lines) per page
    • The start page number
    • The report type. i.e. Normal or Excel
    • If an Excel report is selected the separator character can be entered.
    • If the report can supply an additional totals report, a Totals Only button is displayed.

To print out any help page:
Right click on the page and then select print from the displayed box
